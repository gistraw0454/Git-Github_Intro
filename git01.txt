버전
	프로그램이나 파일이 수정된 사항이 있으면, 버전이 업데이트 되었다.

버전관리시스템
	ex) git 수많은 버전을 쉽게 관리하도록 도와주는 도구

git
	1. 분산형 버전관리 시스템......(저장소==레포지토리)
		원격저장소(github)와 로컬저장소(개인컴퓨터)
		->인터넷연결 없어도 작업가능
	2. 원격저장소에있는 코드를 공유가능
		->효율적 협업가능
	3. 파일들의 수정, 삭제, 추가 등에 대한 모든 변경사항이 기록됨.
	4. 오픈소스(내가만든코드를 공짜로 공개)를 보며 코드의 지식을 넓힐 수 있다.

git 시작
	로컬레포지토리 설정
	마우스 우클릭 ->폴더내에서 Git Bash Here -> git init
		(cmd-> :D -> git init도 가능..)
		git init ?
			.git 이라는 숨김 폴더가 만들어지면서 해당 폴더를 로컬저장소로 사용할 수 있게 됨.
