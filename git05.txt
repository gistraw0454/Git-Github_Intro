git remote add 원경저장소별칭 원격저장소주소
	해당 주소의 원격저장소를 별칭으로 추가(연결)

git remote
	연결시킨 원격 저장소 별칭확인

git remote -v
	연결시킨 원격 저장소 별칭과 주소 확인
git remote rename 원래별칭 바꿀별칭 : 별칭 바꾸기
git remote show 별칭 : 상세정보확인

git log 
	커밋내용(로그)를 확인할수있음
	각각의 커밋내용은 고유의 ID값을 갖고있다
	각각의 id는 중복되지않으므로 id값으로 커밋 구분한다.

git clone 원격저장소주소
	원격저장소의 커밋내용, 수정사항 등 모든 과정 복제

<원격저장소의 내용을 다운로드 하는 방법>
git pull 
	외부에서 내 원격 저장소를 변경한 경우 내 로컬저장소로 커밋하고싶을때
	원격저장소의 최신 커밋상태를 로컬 저장소로 다운로드
	바로 업데이트 시켜버리는거라서 문제가 생길수도있음

git fetch 원격저장소주소
	commit된 내용'만' 복사해 가져와서 임시 저장함
	git merge origin/main 해서 합쳐주면됨.
	
git merge 원격저장소별칭/브랜치이름
	내려받은 커밋을 로컬저장소에 적용
	
git remote rm 원격저장소별칭
	원격저장소 별칭 삭제
